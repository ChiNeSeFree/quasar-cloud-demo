# Quasar Advanced Starter App

This is an app that you can use to learn more about the "Quasar Method". There may be other ways to do things, but this stuff "Works for us"Â®.

## How to use
```
$ npm install -g quasar-cli
$ git clone https://github.com/mstaack/quasar-advanced-starter.git
$ cd quasar-advanced-starter
$ npm install
$ quasar dev
```

## Features
- dev & prod api endpoints
- Vuex stores
- Axios
- Vuelidate
